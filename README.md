# 🎨 AI Virtual Canvas

Gesture controlled drawing app using computer vision.

## 🚀 Features
- ✋ Draw using hand gestures
- 🎨 Real-time color selection
- 🖌️ Smooth drawing experience

## 🧠 Technologies
- Python
- OpenCV
- MediaPipe

## 💡 About
I developed this project to explore computer vision and create an interactive drawing experience.
